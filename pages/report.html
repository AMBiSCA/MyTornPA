Common Issues & Troubleshooting:

Before reporting, consider these common solutions:

  • API Key Errors: Ensure your Torn API key is correct and has "Public" access enabled in Torn settings.
  • Page Not Loading/Blank: Try clearing your browser cache (Ctrl+Shift+R or Cmd+Shift+R) and refreshing.
  • Data Not Refreshing: Check your internet connection. Some data updates automatically, others might require a page refresh.
  • Browser Compatibility: Use modern browsers like Chrome, Firefox, or Edge for best performance.

If issues persist, please use the contact methods above.

Feature Suggestions:

Have an idea for a new tool or improvement? We'd love to hear it!

When suggesting features, please include:

  • Clear Description: What is the feature?
  • Why it's Useful: How would it benefit players or the faction?
  • Examples (Optional): How might it look or function?

While we can't implement every suggestion, all feedback is valuable!
