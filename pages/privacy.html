Privacy Policy

Last Updated: July 18, 2025

What We Collect & Why:

  • Account: Email for login (Firebase Auth).
  • Profile: Display name, Torn Profile ID, & API Key (stored in Firebase Firestore).
  • Torn Data (via API Key): Fetched directly from official Torn City API (Your Torn ID, Name, Faction ID, stats, faction details). API Key is securely stored and used by your browser for direct API calls; we do not share it.
  • Usage Data: Anonymous, aggregated data on feature use to improve MyTornPA.
  • Chat Messages: In-app chat messages stored in Firebase Firestore, visible to participants.

How We Use & Secure Information:

  • Use: To manage account, provide tools (stats, tracking, chat), improve features, & ensure security.
  • Storage: Data stored securely in Google's Firebase Firestore (encryption applies).
  • Security: API Key is securely stored & used only by your browser for Torn API calls; we do not share or sell your personal data.

Policy Changes:

  • Policy may be updated; significant changes will be posted here. Continued use implies acceptance.

Third-Party Services & Your Rights:

  • Google Firebase: For authentication and database.
  • Torn City API: For fetching game data.
  • PayPal: For optional donations (no payment info collected directly).
  • Your Rights: Access, correct, or delete your data (via profile settings or contact us). You can withdraw consent (e.g., by removing your API key).
  • Contact Us: For questions about this policy or data practices, email Support.Tcpa@gmail.com.
